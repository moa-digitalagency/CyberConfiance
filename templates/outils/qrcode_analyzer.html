Qu'est-ce que le Quishing ?

Le Quishing (QR + Phishing) est une attaque où des QR codes malveillants redirigent vers des sites de phishing. Ne scannez jamais un QR code sans vérification préalable.

À Propos de Cet Outil

Comment fonctionne l'analyseur ?

Notre outil analyse les QR codes de manière sécurisée en plusieurs étapes :

  1. Extraction : Le QR code est décodé pour extraire son contenu (URL, texte, etc.)
  2. Analyse des redirections : L'outil suit la chaîne de redirections HTTP sans ouvrir les pages
  3. Détection JavaScript : Les redirections côté client sont identifiées dans le code source
  4. Vérification blacklist : Les URLs sont vérifiées contre les bases de données de menaces
  5. Analyse de patterns : Détection de mots-clés suspects et de techniques de phishing

Pourquoi utiliser cet outil ?

Protection Quishing

Évitez les attaques par QR codes malveillants dans les espaces publics, emails ou documents.

Analyse Sans Risque

L'URL n'est jamais ouverte ni exécutée, évitant tout risque d'infection ou de tracking.

Transparence Complète

Visualisez toute la chaîne de redirections et découvrez la vraie destination du lien.
