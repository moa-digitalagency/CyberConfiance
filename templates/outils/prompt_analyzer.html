À Propos de Cet Outil

Types de Menaces Détectées

Injection de Prompt

Détection des tentatives de manipulation comme "ignore previous instructions", jailbreaks, etc.

Code Malveillant

Identification de code Python, JavaScript ou commandes système dangereuses (eval, exec, etc.).

Obfuscation

Détection de caractères invisibles, encodage base64, et techniques de masquage.

Injection de Rôle

Tentatives de redéfinition du rôle système [admin], [developer], etc.

Recommandations de Sécurité

  • Validez toujours les entrées utilisateur avant de les envoyer à un modèle IA
  • Implémentez des filtres de contenu côté serveur
  • Limitez les capacités d'exécution de code dans vos applications IA
  • Surveillez les logs pour détecter les patterns d'attaque
  • Gardez vos systèmes IA isolés des données sensibles
